## Summary

This source capture was saved as a comparison input, not as a product endorsement. Its main value is that it sharpens the boundary between a broad memory product and a filesystem-first work-memory system.

- Mem0 is useful here as a named comparison target for memory products that emphasize remembering users, applications, and past interactions.
